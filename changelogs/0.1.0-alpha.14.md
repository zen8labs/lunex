# Changelog - 0.1.0-alpha.14

## Features

### Window Management

- **Window auto-maximize on startup**
  - Application window now automatically maximizes when opened (not fullscreen)
  - Ensures optimal screen space utilization on launch
  - Window can still be minimized or restored as normal

## Fixes

### UI/UX

- **Fixed topbar gap issue on macOS**
  - Removed small gap at the top of the window when maximized
  - Added CSS reset for html, body, and root elements to eliminate spacing
  - Window now properly fills the screen edge-to-edge when maximized

## Improvements

### Window Configuration

- Added `maximized: true` to window configuration in `tauri.conf.json`
- Enhanced window positioning logic for macOS to ensure proper top alignment
- Improved CSS layout to prevent unwanted margins/padding

## Technical Details

### Files Changed

- `src-tauri/Cargo.toml`: Updated version to 0.1.0-alpha.14
- `src-tauri/tauri.conf.json`: Added `maximized: true` and updated version
- `src-tauri/src/lib.rs`: Added window maximize logic on startup with macOS position fix
- `src/index.css`: Added CSS reset for html, body, and #root to remove gaps
- `package.json`: Updated version to 0.1.0-alpha.14
