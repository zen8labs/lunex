# Changelog v0.1.0-alpha.32

## Fixes

- Fix "Workspace settings not found" error by lazy initializing settings in backend.
- Fix issue where new chats couldn't be created in a fresh workspace.
- Fix bug where switching chats could cause the recent chat to disappear from the sidebar.
- Improve streaming stability by ensuring `lastMessage` is updated immediately.

## Improvements

- Automatically link the first LLM connection to the workspace during onboarding.
- Move chat ID generation to UUID for better uniqueness and consistency.
- Standardize JSON formatting in Tool Call UI for better readability.
- Enhance frontend state validation for more resilient message sending.
