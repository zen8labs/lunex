# Changelog 0.1.0-alpha.23

## Features

- Implement Image Preview Modal for better visualization of attached images.
- Support editing messages with media attachments.
- Centralize Model Capabilities in the Rust backend for more consistent model behavior (tool support, thinking mode).

- Fix Image Preview and Upload lifecycle to prevent memory leaks.
- Fix Local Image Display using Tauri's file system API.
- Fix Update button error by updating the updater endpoint logic for pre-releases.

## Improvements
