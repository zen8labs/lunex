# Changelog 0.1.0-alpha.17

## Features

- **LLM**: Centralized model capabilities detection in the backend.
- **LLM**: Backend now acts as the single source of truth for `supportsTools` and `supportsThinking` flags.
- **LLM**: Added support for identifying thinking capabilities in `gpt-oss` and `gpt-o1` models.
- **LLM**: Improved model ID parsing to handle provider prefixes (e.g., `openai/`) correctly.

## Improvements

- **UI**: Simplified `ChatInput` logic by removing hardcoded model capability checks.
- **DX**: Added database migration to force refetch of model capabilities after schema changes.
- **DX**: Added debug logging for model capabilities in both backend and frontend.
