# Changelog 0.1.0-alpha.39

## Features

- **Hub**: Thêm hỗ trợ cho loại server MCP `http-streamable` và `streamable-http`.
- **MCP**: Cho phép cấu hình GitHub Copilot MCP trực tiếp từ Hub.

## Improvements

- **MCP Client**: Hỗ trợ alias `streamable-http` cho transport streamable HTTP.
- **Documentation**: Cập nhật mô tả các loại transport MCP được hỗ trợ trong code.
