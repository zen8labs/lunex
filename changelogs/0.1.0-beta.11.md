# Nexo v0.1.0-beta.11

## Features

- **Flow Editor Enhancements**:
  - Added node resizing capability for Group, Rich, and Simple nodes with a new NodeResizer component
  - Implemented drag-and-drop functionality to move nodes into and out of groups
  - Introduced DatabaseNode for schema visualization and editing with SchemaField component
  - Added standardized property editing UI for flow nodes with dedicated property components
  - Improved node parenting logic and visual separators for database nodes

- **Hub & MCP Integration**:
  - Migrated Hub community sections data fetching to RTK Query for better performance
  - Implemented RTK Query for fetching and refreshing hub MCP servers
  - Extracted MCPServerCard component for better code organization
  - Relocated prompt types and community prompts section to the hub feature

- **Settings UI Improvements**:
  - Enhanced settings UI with a new sidebar and lazy loaded sections
  - Added dedicated UI components for workspace settings sections (basic, advanced, model, MCP, and danger zone)
  - Centralized settings constants for better maintainability

- **Chat UI Optimizations**:
  - Implemented textarea auto-resize for better user experience
  - Optimized chat message rendering performance with CSS containment
  - Extracted message controls and attachments into dedicated components
  - Centralized message list state management with new hooks

- **LLM Connection Management**:
  - Refactored LLM connection management to use direct API calls and new UI components
  - Split useAddons hook into granular hooks (useAddonConfig, usePythonRuntime, useNodeRuntime)

## Improvements

- Reorganized flow components into a dedicated `flow` directory with a new `nodes` subdirectory
- Overhauled README with a new logo, condensed sections, and simplified installation instructions
- Improved code organization and maintainability across multiple features

## Bug Fixes

- Fixed various UI/UX issues in flow editor and chat interface
