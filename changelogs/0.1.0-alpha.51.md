# Changelog v0.1.0-alpha.51

## Features

- **Flow Editor**: Bổ sung tính năng **Node Group** cho phép nhóm các node lại với nhau.
  - Tích hợp **NodeResizer** để thay đổi kích thước nhóm linh hoạt.
  - Hỗ trợ tùy chỉnh **Background Color**, **Border Color** và đặc biệt là **Background Alpha** (độ trong suốt nền).
  - Tối ưu hóa phân lớp (Layering) đảm bảo Group luôn nằm dưới các node chức năng.
  - Tối ưu hóa tương tác chuột bằng `pointer-events`, cho phép thao tác với node con xuyên qua nền của nhóm.

## Improvements

- Tinh chỉnh giao diện bảng thuộc tính (Property Panel) để hỗ trợ các điều khiển mới cho Group Node.
- Cập nhật thư viện node mẫu chuẩn.
